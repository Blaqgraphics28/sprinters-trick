### Contribution Guide 💡

- clone from main branch 👨‍💻
- run **npm i** in terminal
- checkout to a new branch
    NOTE: The name of the branch you checkout to must be prepend with your name. **E.g amidsts/login**
- Implement new features 🛠
- Commit your changes 📝
- Push to your branch ⛓
- Raise a pull request 🚀
- submit for review 🧐
